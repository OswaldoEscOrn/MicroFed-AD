# Federated-Edge-AI-for-Real-Time-Environmental-Anomaly-Detection-in-IoT-Networks
This paper extends the original conference work on federated edge AI for environmental anomaly detection by incorporating detailed methodological enhancements tailored for practical IoT deployments.

There are 12 experiments in total, 6 folders with model trainings and each of the folder has 2 experiments.
6 folders :30% Virtual 70% Real Database, 70% Virtual 30% Real Database, 30% anomaly ratio dataset for model training, 70% anomaly ratio dataset for model training, Edge-Machine-Learning-Models and virtual dataset models
Data-Preprocessing is for Edge-Machine-Learning-Models. Except for this folder, other 5 folders mentioned above have corresponding data preprocessing procedures in each of their folders.

Data Folder: all the original datasets gathered from Kaggle.com
